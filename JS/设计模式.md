### 策略模式

目的：减少对 if/else 的写法 使得代码更加优雅具备扩展性

思路：

1. 设计出一个对象 
2. 对象的属性是 if/else 的条件判断

```javascript
/* 设计策略模式对象 */
const typeName = {
	'xiaoming': () => {
		console.log('xiaoming')
	},
	'xiaohong': () => {
		console.log('xiaohong')
	},
	'chuguo': () => {
		console.log('chuguo')
	}
}

/* 方法的调用 */
const fun = () => {
	type && typeName[type] && typeName[type]()
}
```
