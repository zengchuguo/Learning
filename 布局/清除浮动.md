### 背景

​	浮动原理：在当前行浮动起来 使得常规流（例如文字等等）在它周围

#### 解决方案

##### 利用`clear`

​	受到影响的元素添加样式

```css
div{
	clear:left;
}
```

##### **在父元素标签结束前清除浮动**

​	原理和上面一致 使得后面元素添加样式

##### **伪元素清除**

​	原理同上 本质上也是添加元素

```css
.parent:after {
    content: '.';
    height: 0;
    display: block;
    clear: both;
}
```

##### **`overflow` 形成 `BFC` 清除**

​	具体元素同 `BFC`
