### 响应式布局

```java
@media screen and (max-width:XXXpx){
    body{
		样式显示
    }
}
```

### em

相当于当前对象内文本的字体尺寸

### rem

根据根HTML得到font-size得到 `1rem = font-size`

### vw vh

| vw   | 相对于视窗的宽度，视窗宽度是100vw |
| ---- | --------------------------------- |
| vh   | 相对于视窗的高度，视窗高度是100vh |
| vmin | vw和vh中的较小值                  |
| vmax | vw和vh中的较大值                  |
