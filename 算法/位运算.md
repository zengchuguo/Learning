#### [190. 颠倒二进制位](https://leetcode-cn.com/problems/reverse-bits/)

颠倒给定的 32 位无符号整数的二进制位。

```js
/**
 * @param {number} n - a positive integer
 * @return {number} - a positive integer
 */
var reverseBits = function (n) {
  let ans = 0
  for (let i = 0; i < 32 && n > 0; i++) {
    ans |= (n & 1) << (31 - i)
    n >>>= 1
  }
  return ans >>> 0
}
```
